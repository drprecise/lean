Request: no params, no auth

Pairs: 452 (354 crypto, 60 stocks/indices/commodities, 38 forex)
Groups: 12
Fee schedules: 12 (by fee index)

================================================================================
SECTION 1: Publicly captured endpoint shapes and normalization/trade-planning fields
================================================================================

Endpoint: GET https://backend-arbitrum.gains.trade/trading-variables
Auth: none
Returns: JSON object with top-level keys:
  pairs[], groups[], fees[], pairInfos{}, liquidationParams{}, collaterals[],
  feeTiers{}, globalTradeFeeParams{}, borrowingFeePerBlockCap{},
  maxGainP, maxNegativePnlOnOpenP, tradingState, isForexOpen, isStocksOpen,
  isIndicesOpen, isCommoditiesOpen

--- pairs[] (one per tradeable pair) ---
  from              string    base asset symbol (e.g. "BTC")
  to                string    quote asset symbol (always "USD")
  spreadP           string    spread percentage (scaled 1e10, "0" = dynamic)
  groupIndex        string    index into groups[] for category/leverage
  feeIndex          string    index into fees[] for fee schedule

--- groups[] (12 groups: crypto, forex, stocks-1/2/3, indices, commodities-1/2, forex-minor, forex-exotic, altcoins, crypto-degen) ---
  name              string    group name
  minLeverage       string    minimum leverage (scaled 1e3, e.g. "1100" = 1.1x)
  maxLeverage       string    maximum leverage (scaled 1e3, e.g. "200000" = 200x)

--- fees[] (by feeIndex) ---
  totalPositionSizeFeeP       string    total fee on position size (scaled 1e10)
  totalLiqCollateralFeeP      string    liquidation collateral fee (scaled 1e10)
  oraclePositionSizeFeeP      string    oracle fee on position size (scaled 1e10)
  minPositionSizeUsd          string    minimum position size in USD (scaled 1e3)

--- liquidationParams.pairs[] (per pair index) ---
  maxLiqSpreadP               string    max liquidation spread
  startLiqThresholdP          string    liquidation threshold start
  endLiqThresholdP            string    liquidation threshold end
  startLeverage               string    leverage at which liq threshold starts
  endLeverage                 string    leverage at which liq threshold ends

--- pairInfos.maxLeverages[] (per pair index) ---
  value             integer   per-pair max leverage override (0 = use group default)

--- pairInfos.pairFactors[] (per pair index) ---
  protectionCloseFactor       string    protection close factor
  cumulativeFactor            string    cumulative factor

--- globalTradeFeeParams ---
  referralFeeP      string    referral fee portion (scaled)
  govFeeP           string    governance fee portion
  triggerOrderFeeP  string    trigger/limit order fee
  gnsOtcFeeP        string    GNS OTC fee
  gTokenFeeP        string    gToken fee

--- borrowingFeePerBlockCap ---
  minP              string    minimum borrowing fee per block
  maxP              string    maximum borrowing fee per block

--- feeTiers ---
  tiers[]           array     fee discount tiers by points threshold
    feeMultiplier   string    fee multiplier (e.g. "975" = 97.5%)
    pointsThreshold string    points needed for tier

--- collaterals[] ---
  (array of supported collateral token configs)

--- market state flags ---
  tradingState      integer   global trading state
  isForexOpen       boolean   forex market hours status
  isStocksOpen      boolean   stock market hours status
  isIndicesOpen     boolean   indices market hours status
  isCommoditiesOpen boolean   commodities market hours status
  maxGainP          integer   maximum gain percentage (1000000 = 1000%)
  maxNegativePnlOnOpenP integer max negative PnL on open

================================================================================
SECTION 2: Additional fields required for trade execution and position-view
================================================================================

For opening a position (on-chain transaction):
  pairIndex         uint      index of the pair in pairs[]
  long              bool      true = long, false = short
  collateralAmount  uint      collateral in collateral token units
  leverage          uint      leverage multiplied by 1e3
  tp                uint      take profit price (0 = none)
  sl                uint      stop loss price (0 = none)
  openPrice         uint      desired open price (for limit orders)
  orderType         uint      0 = market, 1 = limit

For closing a position (on-chain transaction):
  index             uint      position index for the trader
  pairIndex         uint      pair index

For viewing positions:
  trader            address   wallet address of the trader

For order book / limit orders:
  triggerOrderFeeP from globalTradeFeeParams applies

================================================================================
SECTION 3: Discoverability of execution/position fields
================================================================================

Publicly discoverable (from /trading-variables, no auth):
  - All pair definitions, group leverage ranges, fee schedules
  - Liquidation parameters per pair
  - Per-pair max leverage overrides
  - Borrowing fee caps
  - Fee tier discounts
  - Market hours status (forex/stocks/indices/commodities)
  - Global trade fee breakdown
  - Max gain percentage, max negative PnL on open
  - Collateral token configs

Wallet/user-specific (requires connected wallet address):
  - Open positions (trader address required)
  - Position P&L, funding fees accrued
  - Fee tier level (based on user's accumulated points)
  - Pending limit/trigger orders

Not publicly discoverable from this endpoint:
  - Real-time oracle prices per pair (available via separate price feed/oracle)
  - Historical trade data per user
  - Exact borrowing fee per pair at current utilization (derivable from on-chain state)
